Create `apps/backend/main.py`:
```python
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

app = FastAPI(title="InvestIQ API")

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:3000"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.get("/health")
def health():
    return {"status": "ok", "service": "investiq-api"}
```

Create `apps/backend/auth.py` with `get_current_user` FastAPI dependency:
- Uses `HTTPBearer` security scheme
- Decodes JWT with `PyJWT` using `SUPABASE_JWT_SECRET` env var, algorithm HS256
- Sets `options={"verify_aud": False}` (Supabase uses "authenticated" audience)
- Raises 401 on expired or invalid token
- Returns decoded payload dict (contains `sub` = user UUID, `email`)
- No roles, no RBAC — any valid JWT is accepted

Create `apps/backend/Dockerfile`:
```dockerfile
FROM python:3.12-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000", "--reload"]
```

Create `docker-compose.yml` at repo root:
```yaml
services:
  backend:
    build: ./apps/backend
    ports:
      - "8000:8000"
    env_file: ./apps/backend/.env
    volumes:
      - ./apps/backend:/app
    network_mode: host
    restart: unless-stopped
```

Note: `network_mode: host` lets FastAPI reach Supabase CLI at `localhost:54321`.

Create `apps/backend/.env.example`:
